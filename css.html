CSS

CSS or cascading style sheets is what makes websites look nice. It controls colors, fonts, spacing, layout, and more.

Basic Syntax

CSS works by selecting HTML elements and applying styles to them. Here's an example:


body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

h1 {
  color: darkblue;
  text-align: center;
}

Explanation:

  • background-color: lightblue; → sets the page background color.
  • font-family: Arial, sans-serif; → changes the font of the text.
  • color: darkblue; → changes the color of headings.
  • text-align: center; → centers the heading text.
