Les bases


HTML, CSS et JavaScript

Un site web est généralement créé à l’aide de ces trois éléments. HTML est utilisé pour structurer le site, CSS est utilisé pour le styliser et JavaScript donne au tout des fonctionnalités et de l’interactivité.

On peut comparer un site web à une maison : les briques et les poutres représentent le HTML, la peinture, les meubles et les décorations représentent le CSS, et le JavaScript agit comme la plomberie et le câblage électrique.

VS Code

VS Code, ou Visual Studio Code, est l’éditeur de texte le plus couramment utilisé pour écrire du HTML, du CSS et du JavaScript. Pour les besoins de ce guide, il est recommandé de télécharger VS Code.
