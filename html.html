To begin writing your HTML code, create a file on your desktop and drag it onto VS code to create a project. Create a new file called index.html or anything followed by .html. Once you have created said file, generate a boilerplate by typing and pressing tab or enter.


to begin writing and creating stuff on your website you will need HTML tags. Tags are like instructions for a webpage. They tell the computer what each part of the page is, like a paragraph, a heading, or a picture. Most tags come in pairs: an opening tag at the start and a closing tag at the end. The text or content goes in between. For example, to show a paragraph tag as text on the page, you write: 


    <p>this is a paragraph</p>

Examples of other tags:


<h1>Header with the size of 1</h1>
<a>An anchor used for hyperlinks</a>
