HTML

Pour commencer à écrire votre code HTML, créez un fichier sur votre bureau et glissez-le dans VS Code pour créer un projet. Créez un nouveau fichier appelé index.html ou n’importe quel nom suivi de .html. Une fois ce fichier créé, générez un boilerplate en tapant et en appuyant sur Tab ou Entrée.


Pour commencer à écrire et créer des éléments sur votre site web, vous aurez besoin de balises HTML. Les balises sont comme des instructions pour une page web. Elles indiquent à l’ordinateur ce que représente chaque partie de la page, comme un paragraphe, un titre ou une image. La plupart des balises fonctionnent par paires : une balise ouvrante au début et une balise fermante à la fin. Le texte ou le contenu se trouve entre les deux. Par exemple, pour afficher une balise paragraphe sous forme de texte sur la page, vous écrivez : 


    <p>ceci est un paragraphe</p>

Exemples d’autres balises :


<h1>Titre de niveau 1</h1>
<a>Une ancre utilisée pour les hyperliens</a>

Certaines balises n’ont pas besoin de balise fermante, comme un saut de ligne ou une image :


   <br> 
   
   <img src="image.jpg">
