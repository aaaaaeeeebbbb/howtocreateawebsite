HTML,CSS and JavaScript

A website is typically made with these 3 elements. HTML is used to structure the website, CSS is used to style it and JavaScript gives everything functionality/interactivity. You can think of a website like a house where the bricks and beams are the HTML, the paint, furniture and decorations are CSS and Javascript serves as the plumbing and electrical wiring.

VS Code

VS code or Visual Studio Code is the most commonly used text editor for writing HTML, CSS and JavaScript. For the purposes of this guide, it is reccomended to download VS code.
