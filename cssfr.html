CSS

CSS, ou feuilles de style en cascade(Cascading Style Sheets en anglais), est ce qui rend les sites web esthétiques. Il contrôle les couleurs, les polices, les espacements, la mise en page et bien plus.

Syntaxe de base

Le CSS fonctionne en sélectionnant des éléments HTML et en leur appliquant des styles. Voici un exemple : 


body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

h1 {
  color: darkblue;
  text-align: center;
}

Explication :

  • background-color: lightblue; → définit la couleur de fond de la page.
  • font-family: Arial, sans-serif; → change la police du texte.
  • color: darkblue; → change la couleur des titres.
  • text-align: center; → centre le texte des titres.
